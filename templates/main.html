Eyebeast

Eldritch Archival Utility

    Eyebeast is an archival utility which can help you recover world factbook entiries, regional officers, tags, and flags existed for a region at a given time. Snapshots are taken weekly and automatically pruned after six months. Enter a region name or URL into the search box below to see if we can bring up some records.
