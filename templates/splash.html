Eyebeast

Eldritch Archival Utility

Eyebeast is a data archival utility designed to help recover recently lost data for a given NationStates region. Data is organised into weekly snapshots of the region, featuring a region's flags, banners, search tags, regional officers, and world factbook entries if any were present to be recorded. Snapshots are retained for six months.
